# TLDR Lecture Summaries

Web application that transforms lecture videos into structured, multi-level summaries.

## Overview
TLDR Lecture Summaries was built during the Viscon Hackathon to explore how long-form educational videos can be made more accessible and efficient to consume.

Instead of forcing users to watch full lectures, the system generates **topic-based summaries at different levels of detail**, enabling fast skimming or deep dives depending on user needs.

## Demo
https://tldr.bruol.me

## Features
- **Topic-based structuring** of lecture content  
- **Multi-level summaries**
- **Interactive UI controls** to switch between summary levels  
- **Automatic tagging** for topic exploration  
- Access to original **video and transcript**

## How It Works
1. Lecture videos are processed and transcribed  
2. Content is segmented into topics  
3. Each topic is summarized into multiple levels of detail  
4. Results are presented in an interactive web interface
